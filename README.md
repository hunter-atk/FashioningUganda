# fashioning-uganda-web

> Fashioning Uganda Website

## Prerequisites

- Node 8
- npm 5

## Install/Use Node 8

```
nvm install 8
nvm use 8
```

## Install Dependencies

```
npm install
```

## Run Dev Server

```
npm run start:dev
```
